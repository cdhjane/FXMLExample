 1.Downloading SQLite from URL https://www.sqlite.org/download.html
  For Widows OS，Select "Precompiled Binaries for Windows", download two zip files as below:
  sqlite-dll-win64-x64-3300100.zip
  sqlite-tools-win32-x86-3300100.zip
2.create folder "c:\SQLite", unzip two zip file on this folder
  We can get sqlite3.def、sqlite3.dll and sqlite3.exe  files
3.run SQLite3.exe
4.Download sqlite-jdbc-3.23.1.jar from URL https://bitbucket.org/xerial/sqlite-jdbc/downloads/
5.Open the project in Netbeans IDE.Upon opening the project,right click the Libraries file, and select "Add .jar".
  Select the sqlite-jdbc-3.23.1.jar and import it to the project.
